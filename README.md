# Diabetes-Prediction-API
This is a simple Machine Learning project. It predicts diabetes using a Random Forest model and serves it via a Flask API. The project uses the Pima Indians Diabetes dataset (`diabetes.csv`).

## Files
- `model_building.py`: Trains and saves the model.
- `app.py`: Runs the Flask API.
- `test_api.py`: Tests the API with sample data.
- `diabetes_model.pkl`: Saved model file (generated).
- `diabetes.csv`: Dataset used.
